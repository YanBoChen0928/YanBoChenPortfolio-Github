    CS 碩士學位

    2019年，我來到目前工作診所、主要因為這裡相對過去醫院業務可以給我比較多的時間。一開始我計劃到海外進修醫療資訊領域、因此我從資料分析與管理入手。我參加了南部 AI 經理人班，首次接觸了 Python ，也了解當時企業在資訊應用與管理方面的需求。同時，我與工程師合作完成了台灣疼痛雜誌的線上投稿流程與線上雜誌網站。

    2020年夏天，我在高雄中山大學修習統計學和 R 語言，參考 John Z. 的著作《軟體開發人員職涯發展手冊》並與工程師同學及新朋友討論 CS 領域對我的吸引力與實際面考量，我將焦點轉向程式設計和後端開發，目前先以 Web developer (back-end) 為初始目標，之後再考慮往 software developer 或者 cybersecurity 去深入。

    由於 2019.11-2021.06 家人罹患癌症，在中山大學修完課後，我主要以工作跟陪同家人為主、朝工程師努力的計畫在 2021.09 後才又重新開始。

    2022.03 完成了 GRE 考試，並在陽明交通大學修習了線性代數，在北科大學修習了 C 語言學分。

    2023.06 完成 IELTS 考試。在 2023.10 ，我錄取加拿大溫哥華 2024 年秋季的 Align CS master program。

    2023.11 我開始尋找 bootcamp ，並注意到了 WeHelp 。觀摩過 WeHelp 過往的轉職班成果，以及受到彭彭的 YouTube 前端課程清晰扼要地講解與設計所吸引，聽完 2024.02.03 的直播說明後，我渴望加入這個團體。
